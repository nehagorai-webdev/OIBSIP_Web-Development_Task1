# 🚀 Space Explorer Landing Page

This is a beginner-friendly landing page project built using **HTML** and **CSS**. The website has a space theme with a full-screen background, a heading, a short description, and a button.

## 🛠️ Built With

* HTML5
* CSS3

## ✨ Features

* Space-themed design
* Responsive layout
* Simple and clean interface
* Easy to customize

## 📚 What I Learned

* Creating web pages with HTML
* Styling with CSS
* Using Flexbox for layout
* Making a landing page responsive

## 🚀 How to Run

1. Open the project folder in VS Code.
2. Open `index.html`.
3. Run with Live Server or open it in your browser.

This project was created to practice basic web development skills and improve my understanding of HTML and CSS.
